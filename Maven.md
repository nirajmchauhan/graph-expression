# How to use GExp with Maven #

## step 1: add repository path ##
```
    <repositories>
        <repository>
            <id>mromll</id>
            <url>http://mromll.googlecode.com/svn/repo/</url>
        </repository>
    </repositories>
```

## step 2: add dependency ##
```
        <dependency>
            <groupId>com.myml</groupId>
            <artifactId>gexp</artifactId>
            <version>0.1</version>
        </dependency>
```
